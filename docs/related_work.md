# Related Work

> **參考資料 (Reference Context):** [連結至文獻探討對話紀錄 / Perplexity AI](https://www.perplexity.ai/search/2bf7eda8-7666-45d9-8312-b1abf824477f#14)

本節將回顧與本研究相關之文獻與發展，主要涵蓋四個核心領域：傳統資產配置之侷限、深度強化學習在投資組合最佳化的應用、聰明錢概念 (SMC) 的量化探討，以及與 AI 交易系統整合之微服務架構。透過比較不同方法的貢獻與限制，確立本研究結合 PPO 與 SMC 於多資產動態配置的理論價值與實務意義。

## 1. 傳統資產配置與其侷限性

在投資組合最佳化領域，Markowitz (1952) 提出的均值–變異數模型 (Mean-Variance Optimization, MVO) 奠定了現代資產配置的理論基礎。該模型假設資產報酬服從常態分配，透過期望報酬與共變異數矩陣在風險與報酬之間取得平衡。然而，後續多項研究指出，此類模型在面對非平穩市場或結構性轉變 (regime shift) 時，往往表現不佳。[cite:Markowitz, 1952]

根據在同一套回測框架下的綜合評估顯示，傳統 Markowitz 策略在歷史回測中雖然能取得約 12.3% 的年化報酬，但其最大回撤 (Max Drawdown, MDD) 高達約 28.5%，夏普比率 (Sharpe Ratio) 僅約 0.78，在面對極端事件時暴露出較大的尾部風險。大型機構 (如 PIMCO) 近期報告亦指出，在當前充滿不確定的利率與通膨環境中，傳統的固定比例（如 60/40 股債平衡）或風險平價策略已難以有效控制風險，單一資產或指標曝險過高，往往無法及時應對市場結構的急劇變動。這突顯了對更具彈性、能自適應市場條件的動態風險配置框架的迫切需求。

## 2. 深度強化學習於資產配置之發展

為克服傳統靜態模型的限制，近年來深度強化學習 (Deep Reinforcement Learning, DRL) 被廣泛引入金融資產配置領域。有研究比較了 LSTM、Transformer、DDPG 等深度學習演算法，指出強化學習在處理非線性關係與非平穩市場環境方面具備顯著優勢。其中，近端策略最佳化 (Proximal Policy Optimization, PPO) 演算法因其策略更新穩定性與較高的樣本效率，成為連續動作空間配置的主流。

實證數據支持了 DRL 的優越性：有研究在股票投資組合最佳化實驗中發現，PPO 策略在允許做空情境下可達約 22.18% 的年化報酬與 1.10 的夏普比率，最大回撤約控制在 −17.21%，相較於等權重 (MDD 約 −21.36%) 或市值加權基準具有顯著優勢。此外，台灣的相關碩論（如中央大學孟繁淳的 DDPG/PPO 資產配置研究；政治大學 PPO 模型研究，2019）在設計投組模型時，明確將平均報酬、波動度、最大回撤等風險指標納入獎勵函數 (Reward Function)，證實適當的獎勵設計能在相同風險水準下提升報酬。然而，這類研究多數依賴傳統技術指標與歷史價格作為觀測狀態 (State)，缺乏對機構流動性結構的直接刻畫，容易隨市場假突破遭受損失。

## 3. 聰明錢概念 (SMC) 與市場結構量化之探討

聰明錢概念 (Smart Money Concepts, SMC) 在實務交易社群逐漸成為解析機構行為的核心框架。實務經驗與分析指出，大量散戶依賴 RSI、MACD 等「滯後指標」，往往在價格完成主要位移後才發出訊號，成為約 90% 被機構利用的流動性來源。相對地，SMC 主張透過辨識市場結構斷裂 (Break of Structure, BOS)、性格轉變 (Change of Character, CHoCh)，以及流動性區域、訂單區塊 (Order Block, OB) 與公平價值缺口 (Fair Value Gap, FVG)，提前尋找具流動性支撐的合理入場區間。

在加密貨幣與外匯市場中，實務界已能透過鏈上數據或特定歷史訊號量化聰明錢的勝率與累積報酬，證實機構視角能有效提升進場精準度與減少盲目追價。從傳統 COT 分析出發，「大額投機者與散戶淨多單差異」已被證明與指數未來報酬高度相關。然而，目前 SMC 大多停留在主觀裁量交易或受限於半自動化指標提示，學術界鮮少有研究系統性地將 SMC 核心特徵（如 FVG 距離百分比、OB 觸碰狀態）量化為連續狀態向量並整合入強化學習環境。本研究即針對此研究缺口，將機構流動性指標正式納入 PPO 模型的觀測狀態，以彌補單純依賴技術指標的不足。

## 4. AI 交易系統架構與微服務戰情室

除了演算法設計，AI 模型在金融機構的實際落地亦需仰賴堅實的系統架構。相關產學界實務（如 LinkedIn 分散式交易平台、臺大的自適應風險投組平台研究）指出，企業端引入 AI 投資時，資料治理層與系統解耦至關重要。利用 Spring Boot 等微服務框架結合事件驅動 (如 Kafka) 機制，不僅能保證交易訂單、撮合服務與資料庫的獨立性與高穩定性，亦能支援非同步的複雜 AI 推論。

此外，為解決強化學習被批評的「黑箱 (Black-box)」問題，建立具備即時監控能力的「戰情室 (War Room)」成為最新趨勢。誠如在智慧工廠與金融風險管理研究中所見，建立基於 React 等前端技術的高互動可視化儀表板，能即時呈現模型的多資產配置權重轉移、SMC 事件觸發與淨值回撤風險。本專案借鑒這些實務成果，提出一套整合微服務後端與戰情室前端的完整架構，以此支援 PPO 代言人策略，確保所提之動態配置模型具備高度決策透明性與實際部署能力。

## 總結

綜合上述文獻可發現，雖有大量研究探討 PPO 在投資組合的應用並取得了超越市值的實證數據，但其觀測特徵仍多屬傳統技術指標；而 SMC 雖能有效捕捉機構流動性以降低假突破風險，卻極少被整納入自動化最佳化框架。本研究旨在填補此空缺：將 SMC 量化特徵結合 PPO 的動態配置能力，同時透過微服務戰情室確保系統的穩定性與透明度，為面臨高波動與極端風險的現代金融市場，提供一個兼具創新性與工程實踐性的多資產配置解決方案。
